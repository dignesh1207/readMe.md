## 🚀 Featured Projects

### ☁️ [CloudVPN — Self-Hosted Multi-Location VPN](https://github.com/dignesh1207/selfhosted-vpn)

> A production-grade, free self-hosted VPN across **AWS EC2 (Mumbai)** and **Google Cloud (Montreal)** with a real-time control center dashboard, Chrome extension, and Pi-hole ad blocking.

**What I built:**
- WireGuard VPN servers on 2 cloud providers with full security hardening
- Real-time monitoring dashboard (Node.js + Socket.io + Chart.js)
- Chrome extension (Manifest V3) for one-click location switching
- Pi-hole DNS ad blocking — 554,000+ domains blocked
- UFW firewall + Fail2Ban intrusion prevention
- Cloudflare Tunnel for HTTPS
